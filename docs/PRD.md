## 1. 제품 개요

### 1.1 제품명
**Claude Sticky** — Virtual Office with Team

### 1.2 한 줄 설명
팀이 함께 일하는 가상 사무실에서 업무를 공유, 관리하고 포인트로 보상받는 협업 도구

### 1.3 배경 및 문제 정의

| 문제 | 설명 |
|------|------|
| 팀 과제 투명성 부재 | 누가 무엇을 하는지 한눈에 파악하기 어려움 |
| 원격 협업의 고립감 | 같은 공간에 있다는 소속감 없이 도구만 존재 |
| 낮은 업무 완료 동기 | 완료해도 피드백 늦거나 없음, 성취감 부족 |
| 복잡한 협업 도구 | Jira, Asana 등은 설치·설정 과정이 무거움 |

### 1.4 해결 방법

Gather.town 스타일의 **가상 사무실**에서 팀원들이 캐릭터로 함께 존재하며,
과제 완료 시 **포인트**를 획득하고, **상점**에서 아바타를 꾸밀 수 있는
가볍고 재미있는 팀 협업 도구를 제공한다.

## 3. 대상 사용자

### 3.1 주요 사용자 그룹

| 구분 | 설명 |
|------|------|
| **팀 리더 (Admin)** | 업무 생성·배분, 팀원 관리, 포인트 부여를 담당 |
| **팀원 (Member)** | 업무 수행, 포인트 획득, 아바타 커스터마이징 |
| **체험 사용자** | 데모 계정으로 회원가입 없이 기능 체험 |

### 3.2 페르소나

**팀 리더 — "승원"** (스타트업 PM, 5인 팀 운영)
- 매일 오전 팀 과제 현황을 빠르게 확인하고 싶음
- 팀원들이 업무를 성취감을 느끼며 진행하길 원함
- 복잡한 도구 도입에 부정적 견해

**팀원 — "철수"** (개발자, 재택 근무)
- 내가 오늘 뭘 해야 하는지 명확히 알고 싶음
- 일하는 공간에서 동료와 연결된 느낌을 받고 싶음
- 완료한 과제에 대한 보상과 피드백을 원함

## 6. 비기능 요구사항

### 6.1 성능

| 항목 | 기준 |
|------|------|
| 초기 로딩 | 3초 이내 (LCP) |
| 실시간 위치 동기화 | 100ms 이하 지연 |
| 과제 목록 로딩 | 1초 이내 |
| Canvas 렌더링 | 30fps 이상 |

### 6.2 반응형 / 화면 크기

| 항목 | 기준 |
|------|------|
| 앱 기준 해상도 | 360×560px (콘텐츠 영역) |
| 팝업 창 크기 | 360×720px (브라우저 크롬 포함) |
| 데스크탑 전용 | MVP 범위 (모바일 미지원) |

### 6.3 보안

| 항목 | 기준 |
|------|------|
| 인증 | Supabase Auth (JWT) |
| 세션 | SSR 쿠키 기반, scope: local signOut |
| DB 접근 | RLS(Row Level Security) 적용 |
| Admin 전용 기능 | 서버사이드 role 검증 |

### 6.4 접근성

- 키보드 기반 내비게이션 지원 (WASD 이동, Space 상호작용)
- 색상 외 상태 표시 (아이콘/텍스트 병용)

## 7. 데이터 모델 요약

```
teams
  └── profiles (user)
        └── tasks (assigned_to[])
              └── task_comments
        └── purchases
              └── shop_items
        └── point_logs
        └── notifications
```

### 핵심 테이블

| 테이블 | 설명 |
|--------|------|
| `teams` | 팀 정보, 초대 코드 |
| `profiles` | 유저 프로필, 역할, 포인트, 아바타 파츠 |
| `tasks` | 과제 정보, 상태, 담당자, 포인트 |
| `task_comments` | 과제 댓글 |
| `shop_items` | 상점 아이템 (아바타 파츠) |
| `purchases` | 구매 이력 |
| `point_logs` | 포인트 획득/소비 이력 |
| `notifications` | 실시간 알림 |

## 8. 화면 구성

### 8.1 페이지 흐름

```
[런처 화면] (일반 탭)
    ↓ 앱 실행하기 클릭
[팝업 창 360×560]
    ├── /login  — 로그인 폼
    ├── /signup — 회원가입 폼
    └── / (메인)
          ├── 가상 사무실 (홈)
          │     └── 오브젝트 인터랙션 → 각 패널 오픈
          └── 관리자 전용 영역
```
